# 🤖 MCP Workflow AI

## 🪄 Overview
Building AI agents that can interact with real-world tools via the **Model Context Protocol (MCP)**.
- **Why it matters**: Moves beyond chatbots to "Action Bots" that can manage tasks in GitHub, Jira, and Slack.
